Technology used

HTML

HTML (HyperText Markup Language) structures the content of the web application, providing the backbone for all user interface elements and ensuring accessibility and semantic clarity.

Javascript

JavaScript enables dynamic interactivity, real-time feedback, speech-to-text, and seamless user experience by handling client-side logic and asynchronous communication with the backend.

Machine Learning

Machine Learning powers the spam detection engine, using a trained Naive Bayes model to analyze and classify emails/messages as spam or not, based on learned patterns from real data.

CSS

CSS (Cascading Style Sheets) is used to style and layout the application, ensuring a visually appealing, responsive, and accessible interface across all devices and modes (including dark mode).

Flask

Flask is a lightweight Python web framework that powers the backend, handling routing, model inference, file uploads, and serving the web interface securely and efficiently.

Dataset

The dataset consists of thousands of labeled emails/messages (spam and ham), providing the real-world data needed to train and evaluate the machine learning model for accurate predictions.
